# GokulaHealth 🏥

GokulaHealth is an Android healthcare management application developed using Kotlin and Android Studio. The application is designed to help users manage healthcare-related activities efficiently with a simple and modern user interface.

---

## Features ✨

- User Authentication & Login
- Patient Health Record Management
- Appointment Scheduling
- Notifications & Alerts
- Secure Data Storage
- Modern Android UI
- Fast & Responsive Performance

---

## Tech Stack 🛠️

- Kotlin
- Android Studio
- Jetpack Compose
- Firebase / Room Database
- MVVM Architecture
